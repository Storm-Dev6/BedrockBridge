# Architecture Decision Records

Az ADR-ek immutable döntéstörténetet alkotnak. Státuszuk `Proposed`, `Accepted`, `Deprecated` vagy `Superseded by ADR-NNNN`. Elfogadás után a tartalom nem írható át érdemben; változó döntés új ADR-rel felülírja. A Phase 0.5 ADR-jei jelenleg `Proposed` állapotúak, és együtt fogadandók el az engineering specificationnel.
